# Codex Skills

Personal Codex skills for Unity/Maya/Blender asset workflows.

## Skills

- `fbx-zhengli`: Unity FBX cleanup and material/texture organization.
- `fbx-zhengli-2`: FBX cleanup plus optional Maya quad retopology and texture transfer.
- `image-3d-modeling`: Image-to-3D workflow through Hunyuan 3D, Blender/Maya, and Unity finalization.

## Install

Copy the folders under `skills/` into your Codex skills directory:

```text
C:\Users\youde\.codex\skills\
```

Expected local layout:

```text
C:\Users\youde\.codex\skills\fbx-zhengli\SKILL.md
C:\Users\youde\.codex\skills\fbx-zhengli-2\SKILL.md
C:\Users\youde\.codex\skills\image-3d-modeling\SKILL.md
```
